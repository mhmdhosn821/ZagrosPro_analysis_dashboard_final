## Responsive Behavior

- Dashboard fills entire admin workspace
- Scrollbar appears when content exceeds viewport
- Iframe maintains aspect ratio
- Tabs stack on smaller screens (handled by flex)
- Settings form adapts to container width

## Key Design Principles

1. **Glassmorphism**: Transparency + blur for modern look
2. **Dark on Light**: Dark glass container contrasts with white WP admin
3. **RTL Support**: All text and layout flows right-to-left
4. **Persian Typography**: Vazir font for optimal readability
5. **Accessibility**: Good contrast ratios, clear focus states
6. **Smooth Interactions**: Transitions for better UX
7. **Visual Hierarchy**: Clear distinction between sections
8. **Empty States**: Beautiful messages instead of blank spaces
